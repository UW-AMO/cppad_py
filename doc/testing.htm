Running the Cppad Py Tests

Configuration
You must edit setup.py to configure Cppad Py for your system.

get_cppad.sh
The next step is to get a copy of Cppad using the command
     bin/get_cppad.sh

setup.py
The next step is the build the Python cppad_py module using one of the command:
     python setup.py build_ext --inplace
 where python is the Python executable you will be using with Cppad Py.

check
The next step is to check cppad_py on your system using
     cd build
     make check
check_swig_xam
For lang equal cplusplus, octave, perl, python,
     make check_swig_xam_lang
 only runs a simple test of Swig, for the particular language, on your system.

check_lib
For lang equal cplusplus and python,
     make check_lib_lang
 only runs a the tests for the C++ or Python on your system.
