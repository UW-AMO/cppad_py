Stop Current Recording and Store in an a_fun Object

Syntax
af = cppad_py.a_fun(ax, ay)

ax
This argument must be the same as ax returned by the previous call to independent; i.e., it must be the independent variable vector. We use n to denote the number of independent variables (the size of ax ).

ay
This argument is a numpy array with elements that correspond to c++ a_double. It specifies the dependent variables. We use m to denote the number of dependent variables (the size of ay ).

af
The result is a cppad_py.a_fun object. It has a representation for the a_double operations that mapped the independent variables to the dependent variables. These operations define the function that can be differentiated.

Example
All of the a_fun examples use an a_fun constructor.
